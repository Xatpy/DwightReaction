# Giffing
Get gifs in full-screen
https://xatpy.github.io/giffing/

## Usage
### Searches
/key?GIF_SEARCH
### Controls
- Key R or key A: Activate auto mode, changing gif each 1,5 seconds
- Click or press any other key to change manually to a new random gif
